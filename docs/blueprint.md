# **App Name**: CampusCard

## Core Features:

- Secure User Authentication: Allows students and staff to securely log in using their credentials.
- Digital ID Card Generation: Generates a verifiable digital student ID card upon successful login, and staff ID Card upon successful login.
- User Registration: Allows new users (students and staff) to register their accounts.
- Camera Integration: Integrates with the device camera to take a professional picture for the digital ID card.
- Dashboard: Dashboard where users can create and view their digital ID Card

## Style Guidelines:

- Primary color: Deep Blue (#1E3A8A) for trust and security, reflecting the institution's reliability.
- Background color: Light blue (#E3F2FD), a very light tint of the primary color, offering a clean and unobtrusive background.
- Accent color: Teal (#38A3A5) to draw the user's eye to important ui elements
- Headline Font: 'Space Grotesk', a modern sans-serif for headlines and short descriptions; Body font: 'Inter', a grotesque-style sans-serif for main contents.
- Use clear and recognizable icons for navigation and key actions.
- Clean, intuitive layout with a focus on easy navigation. Use of card-based design to present information clearly.
- Subtle transitions and animations to enhance user experience, like card reveal animations.
